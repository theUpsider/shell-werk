# Description

The chat interface allows the user to select from a list of available LLMs. The LLMs are are retrived based on the settings. The application musst support vLLM as well as ollama. Baed on which LLMs are offered, the user can select one before starting a chat session on the top in a drop down. If he already selected one before, it is preselected and he can start chatting right away.
The settings of the selected LLM as well as the configuration for the proividers need to be persisted across app restarts.
