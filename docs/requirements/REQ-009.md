# Description

The application shall support multiple LLM providers, specifically vLLM and Ollama. The configuration for these providers and the selected LLM settings must be persisted across application restarts. Settings is located under a settings button which opens a modal.

# Acceptance Criteria

- [x] The application supports connection to vLLM.
- [x] The application supports connection to Ollama.
- [ ] The application retrieves the list of available LLMs based on the configured settings.
- [x] The settings for the selected LLM are saved and persisted.
- [x] The configuration for the providers (e.g., URLs, keys) is saved and persisted across app restarts.
- [ ] The settings modal is accessible via a settings button in the chat interface.
