# Description

The application shall implement a dialogue feedback loop that orchestrates the interaction between the user, the AI, and the tools. This loop starts with the user's message and continues through AI planning, tool execution, and result feedback until a final answer is provided.

# Acceptance Criteria

- [ ] The feedback loop is initiated when the user sends a message.
- [ ] The AI plans an action based on the user's input.
- [ ] If the AI outputs a tool call, the application parses and executes the tool.
- [ ] The output of the tool execution is returned to the AI.
- [ ] The AI uses the tool output to plan the next action.
- [ ] The loop continues until the AI provides a final answer.
