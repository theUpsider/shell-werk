# Description

The application shall include a sidebar on the left side of the interface to manage chat sessions. This sidebar allows users to view a list of past chats, select a past chat to load its history, and start a new chat session.

# Acceptance Criteria

- [ ] A sidebar is displayed on the left side of the application window.
- [ ] The sidebar lists past chat sessions.
- [ ] Clicking on a past chat session in the sidebar loads the corresponding chat history into the main chat interface.
- [ ] The sidebar contains a button to start a "New Chat".
- [ ] Clicking the "New Chat" button clears the current chat view and starts a fresh session with an empty history.
- [ ] The sidebar persists the list of chats across application restarts (implied by "past chats").
