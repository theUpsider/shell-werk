title: REQ-010: LLM Selection in Chat
needs: [N-005]

# Description

The chat interface shall allow the user to select the desired LLM from a list of available models. This selection must be made via a dropdown menu at the top of the interface before starting a chat session. If he already selected one before, it is preselected and he can start chatting right away.

# Acceptance Criteria

- [ ] A dropdown menu is available at the top of the chat interface.
- [ ] The dropdown lists all available LLMs retrieved from the providers.
- [ ] The user can select an LLM from the dropdown.
- [ ] The selection must be made before starting a chat session except if a previous selection exists.
- [ ] The selected LLM is used for the subsequent chat session.
- [ ] If the user has previously selected an LLM, it is preselected when the chat interface is opened.
