# Description

The chat interface shall allow the user to select the desired LLM from a list of available models. Selection is a global app default used for all new chats and is preselected on launch. A dropdown at the top of the interface provides the selection.

# Acceptance Criteria

- [ ] A dropdown menu is available at the top of the chat interface.
- [ ] The dropdown lists all available LLMs retrieved from the providers.
- [ ] The user can select an LLM from the dropdown.
- [ ] The selected LLM becomes the global default used for all new chats.
- [ ] If the user has previously selected an LLM, it is preselected on launch.
