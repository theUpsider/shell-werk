# Description

The application shall provide a chat interface that serves as the main interaction point for the user. This interface must include a central chat area for displaying messages and an input bar at the bottom for user input. The design should be intuitive, resembling popular chat applications like ChatGPT.

# Acceptance Criteria

- [ ] The application displays a central chat area that occupies the majority of the screen space.
- [ ] An input bar is fixed at the bottom of the chat interface.
- [ ] The input bar contains a text input field for typing messages.
- [ ] The input bar contains a "Send" button to submit the message.
- [ ] Pressing the "Enter" key within the text input field submits the message.
- [ ] The chat area is scrollable to view previous messages in the current session.
- [ ] The chat area automatically scrolls to the bottom when a new message is added.
