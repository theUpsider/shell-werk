# Description

The application shall provide a chat interface that serves as the main interaction point for the user. This interface must include a central chat area for displaying messages and an input bar at the bottom for user input. The design should be intuitive, resembling popular chat applications like ChatGPT.

# Acceptance Criteria

- [x] The application displays a central chat area that occupies the majority of the screen space. (Covered by `frontend/src/App.requirements.test.tsx`)
- [x] An input bar is fixed at the bottom of the chat interface. (Covered by `frontend/src/App.requirements.test.tsx`)
- [x] The input bar contains a text input field for typing messages. (Covered by `frontend/src/App.requirements.test.tsx`)
- [x] The input bar contains a "Send" button to submit the message. (Covered by `frontend/src/App.requirements.test.tsx`)
- [x] Pressing the "Enter" key within the text input field submits the message. (Covered by `frontend/src/App.requirements.test.tsx`)
- [x] The chat area is scrollable to view previous messages in the current session. (Covered by `frontend/src/App.requirements.test.tsx`)
- [x] The chat area automatically scrolls to the bottom when a new message is added. (Covered by `frontend/src/App.requirements.test.tsx`)
