# Description

The application shall allow developers to configure the visibility of tools in the user interface. Developers can mark a tool as "UI visible," making it toggleable by the user. Tools not marked as UI visible are enabled by default and not shown in the toggle list.

# Acceptance Criteria

- [ ] Developers can mark a tool as "UI visible" in the code/configuration.
- [ ] Developers can provide a short description for the tool.
- [ ] Tools not marked as "UI visible" are enabled by default.
- [ ] Tools not marked as "UI visible" are automatically included in the list of tools given to the LLM.
