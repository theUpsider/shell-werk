# Description

The application shall allow developers to configure the visibility of tools in the user interface. Developers can mark a tool as "UI visible," making it toggleable by the user. Tools not marked as UI visible are hidden but enabled by default and not shown in the toggle list. A global settings page allows disabling non-visible tools.

# Acceptance Criteria

- [ ] Developers can mark a tool as "UI visible" in the code/configuration.
- [ ] Developers can provide a short description for the tool.
- [ ] Tools not marked as "UI visible" are hidden but enabled by default.
- [ ] Tools not marked as "UI visible" are automatically included in the list of tools given to the LLM unless globally disabled.
- [ ] A global settings page allows disabling non-visible tools.
