# Description

For tools marked as "UI visible," the chat interface shall provide a mechanism for users to toggle them on or off. This allows users to control which tools are available to the LLM for a specific chat session.

# Acceptance Criteria

- [ ] UI visible tools are displayed in the chat interface (e.g., in the input bar area).
- [ ] Users can toggle these tools on or off.
- [ ] Hovering over the toggle button displays the tool's description as a tooltip.
- [ ] The state of the toggle determines whether the tool is available to the LLM for the chat session.
- [ ] When Chat Only mode is enabled, tool toggles have no effect and tools remain disabled.
